メルカリ メルカリShops ラクマ PayPayフリマ ヤフオク BASE STORES minne Creema

💻 インストール

  1. Chrome Web Store で「フリマアシスト」を検索
    ID: jcbljdgnpcckiamdgmnfhijgkkaogmgg
  2. 「Chromeに追加」ボタンをクリック
  3. 拡張機能がインストールされる
💡 備考: Chrome専用。Firefox / Edge等では使用不可。Googleアカウントでログインしておくと複数端末で設定共有可能。

🔐 シークレットコード設定（必須）

コピー出品・画像ダウンロード・CSV出力などの機能を使うにはシークレットコードが必要です。

  1. きよまろ氏の公式LINEを友だち追加
  2. LINEで「フリマアシスト」とそのまま7文字だけ送信
  3. 自動返信でシークレットコードが届く
  4. Chrome右上のフリマアシストアイコンをクリック
  5. コード入力欄に入力して【入力】ボタン→「OK」表示で完了
⚠️ 注意:
  • 送信メッセージに余計な文字・記号・スペースが入っていると取得不可
  • シークレットコードは定期的に変更される（LINEで通知あり）
  • 公式LINE登録後すぐブロック / 何度もブロックするとコード取得不可

🔓 裏コード（自動マルチボタン）

自動値下げ・値上げ・出品停止・再出品などの自動処理機能を使うには裏コードが必要。

  1. 公式LINEで「裏コード」と送信
  2. 届いたコードをオプション画面に入力
  3. 自動値下げ金額などはオプション画面でカスタマイズ

⚡ 主な機能

🔄

コピー出品

商品ページをまるごとコピー。別フリマサイトへのクロス出品も対応

📅

日時表示

出品日時・更新日時・コメント投稿日時を商品ページに表示

📷

画像ダウンロード

商品画像を単品または一括でダウンロード

🔍

検索拡張

検索結果に他フリマサイトのワンクリック検索ボタンを追加

💬

定型文機能

よく使うコメントを最大5つ登録、ワンクリック挿入

💲

価格変更ボタン

編集画面で「-100」「-1000」「-5%」等で即座に価格調整

🤖

自動マルチボタン

自動値下げ/値上げ・出品停止・削除・再出品を自動処理

📊

CSV出力

売却済み商品情報をCSVエクスポート。確定申告に活用

⚙️ カスタマイズ

Chrome右上のフリマアシストアイコンをクリックすると設定画面が開きます。

  • 普段使わないサイトのボタンを非表示にできる
  • 自動値下げ金額のカスタマイズ（オプション画面）

⚠️ 注意事項

重要:
  • 短時間に大量操作するとBot判定されアカウント停止のリスクあり（10ページ程度なら問題なし）
  • 売却済みページを削除するとCSVデータ取得不可
  • 本機能利用による損害について開発者は責任を負わない
📱 スマホで使う: スマホアプリ版はないが、Androidの Yandex Browser / Quetta / Edge Canary で利用可能。
💰 無料: LINE登録のみで全機能無料。開発者: きよまろ氏。
