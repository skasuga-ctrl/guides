インストール

1
Chrome Web Store で検索
ID: jcbljdgnpcckiamdgmnfhijgkkaogmgg
2
「Chromeに追加」ボタンをクリック
3
拡張機能がインストールされる
💡Chrome専用。Firefox / Edge等では使用不可。Googleアカウントでログインすると複数端末で設定共有可能。
🔐

シークレットコード設定（必須）

コピー出品・画像DL・CSV出力などの機能に必要。

1
きよまろ氏の公式LINEを友だち追加
2
LINEで「フリマアシスト」とそのまま7文字だけ送信
3
自動返信でシークレットコードが届く
4
Chrome右上のアイコンをクリック
5
コード入力欄に入力→【入力】→「OK」表示で完了
⚠️ 注意事項
  • 送信メッセージに余計な文字・記号・スペースがあると取得不可
  • コードは定期的に変更（LINEで通知あり）
  • 公式LINE登録後すぐブロック / 何度もブロックするとコード取得不可
🔓

裏コード（自動マルチボタン）

自動値下げ・値上げ・出品停止・再出品などの自動処理機能用。

  1. 公式LINEで「裏コード」と送信
  2. 届いたコードをオプション画面に入力
  3. 自動値下げ金額などはオプション画面でカスタマイズ
⚡

主な機能

🔄

コピー出品

商品ページをまるごとコピー。クロス出品対応

📅

日時表示

出品日時・更新日時・コメント日時を表示

📷

画像DL

商品画像を単品または一括でDL

🔍

検索拡張

他フリマサイトのワンクリック検索ボタン

💬

定型文

よく使うコメントを最大5つ登録・ワンクリック挿入

💲

価格変更

-100 / -1000 / -5% などのボタンで即調整

🤖

自動マルチ

自動値下げ/値上げ・停止・削除・再出品

📊

CSV出力

売却済み情報をCSVエクスポート。確定申告に

⚙️

カスタマイズ

Chrome右上のアイコンクリックで設定画面を開く。

  • 普段使わないサイトのボタンを非表示にできる
  • 自動値下げ金額のカスタマイズ（オプション画面）
🚨 重要な注意事項
  • 短時間に大量操作するとBot判定→アカウント停止リスク（10ページ程度ならOK）
  • 売却済みページを削除するとCSVデータ取得不可
  • 本機能利用による損害について開発者は責任を負わない
📱スマホアプリ版はないが、Androidの Yandex Browser / Quetta / Edge Canary で利用可能。
💰LINE登録のみで全機能無料。 開発者: きよまろ氏。
