📦 とりこみ君

商品情報を自動取得してスプレッドシートに転送

Knowledge Base > Tool Guides > とりこみ君

🌐 対応サイト

eBay 楽天市場 Amazon メルカリ ヤフオク PayPayフリマ ラクマ

⚡ 主な機能

👉

ワンクリック抽出

商品名・価格・画像URL・出品者情報を自動取得

📑

複数シート対応

最大10個のシートを登録、用途別に使い分け

🚨

アラート機能

除外キーワード・注目ハイライト・自動警告

⚙️

カスタマイズ

ボタン配置・取得画像数・待機時間を調整

📝 使い方（5ステップ）

  1. Chrome Web Store からインストール
    ID: pibfligoaakdjohpiapdgnldpkoicepb
  2. 設定画面で Google スプレッドシートのURLを登録
  3. 対応サイトの商品ページを開く
  4. 「とりこみ」ボタンをクリック
  5. 商品情報が自動的にスプレッドシートに追加される
💡 Tips: 複数商品の連続取り込みは、ページを切り替えながらボタンをクリックするだけでOK。

🔗 一括シートとの連携

取得した商品情報を一括シートにインポート可能。

  1. リサーチ用スプレッドシートで情報収集
  2. 一括シートにコピー
  3. 価格計算・翻訳を実行
