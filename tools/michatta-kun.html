⚡ 主な機能

📝

自動記録

商品ページ訪問時にURLまたは商品IDで自動保存

🏷️

バッジ表示

検索結果一覧に「閲覧済み」バッジを表示

📊

大容量対応

登録件数無制限（10万件以上もOK）、古いデータは自動削除

✍️

手動登録

URLや商品IDでの手動登録も可能

👤

出品者チェック

出品者情報をワンクリックで確認

🔔

アラート設定

仕入れ判断をサポートするアラート機能

📝 使い方

  1. Chrome Web Store からインストール
    ID: bpeiffjpdepilodkgahpnohpddpppphj
  2. 商品ページを訪問すると自動で記録される
  3. 検索結果画面でバッジを確認
  4. 未チェック商品だけを効率的にリサーチ

💡 活用シーン

  • 仕入れ候補検索時の重複チェック削減
  • 相場調査で既チェック出品の判別
  • 大量リサーチ時の見落とし防止

🔒 プライバシー

安心設計: 全データはブラウザのローカルストレージに保存。外部サーバーへの送信なし。
