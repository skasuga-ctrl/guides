🔧Troubleshooting WIP

Common errors and their solutions.

MCP Connection Issues

Error: MCP server not responding

claude mcp list   # Check server status

Solution: Restart the MCP server process. Check ~/.claude.json for correct configuration paths.

Error: Gemini API rate limit (429)

[429 Too Many Requests] You exceeded your current quota

Solution: Wait 30 seconds or switch to gemini_quick (Flash) instead of gemini_review (Pro). Check quota at ai.dev/rate-limit.

Claude Code Tips

Context window running low

When context reaches ~80%, Claude may start losing earlier conversation details.

Actions:

  • Use /compact to compress context manually
  • Disable unused MCP servers in project settings
  • Delegate large file analysis to gemini_quick
